# Collab — Démarrage rapide

> 1 page. Si t'as besoin de plus, c'est qu'on a raté l'UX.

---

## Sur n'importe quel Wi-Fi (avec ou sans internet)

**Une personne (le host)** :
1. Lance **Collab Host** → clique sur **📡 Héberger**
2. Affiche le QR code à l'écran

**Les autres (clients)** :
1. Scannent le QR avec leur téléphone, ou cliquent le lien partagé
2. C'est tout

→ Vous êtes dans la même room. Notes, fichiers, Q&A en temps réel.

---

## Pas de Wi-Fi du tout ?

Quelqu'un active le **partage de connexion** de son téléphone :
- iPhone : Réglages → Partage de connexion → Activer
- Android : Paramètres → Réseau → Point d'accès Wi-Fi → Activer

Pas besoin d'avoir des données mobile actives. Le hotspot crée juste un Wi-Fi local.

Tout le monde se connecte à ce Wi-Fi → revient à l'étape ci-dessus.

## Problème ?

| Symptôme | Solution |
|---|---|
| Clients ne se connectent pas | Vérifier qu'ils sont sur **le même Wi-Fi** que le host |
| QR code ne s'ouvre pas au scan | Taper l'adresse manuellement (ex: `192.168.1.42:5173`) |
| App ne se lance pas (Windows) | Installer Docker Desktop d'abord (lien fourni à l'install) |
| Antivirus bloque | Ajouter Collab Host en exception |
| Tout est lent à 20+ personnes | Le host doit être un PC, pas un téléphone. Wi-Fi 5GHz mieux. |
